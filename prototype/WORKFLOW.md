# Рабочий контур

Одна точка входа для повседневной работы:

```bash
make all
```

Что делает:

1. Пересобирает HTML-страницы из генераторов.
2. Обновляет минифицированные Tilda-блоки в `prototype/tilda_blocks_min/`.
3. Показывает, что изменилось.

Полезные команды:

- `make build` — только генерация HTML;
- `make minify` — только минификация блоков;
- `make status` — проверка устаревших/отсутствующих min-файлов;
- `python3 prototype/devflow.py help` — справка по CLI.

Где редактировать:

- `prototype/build_course_pages.py` — курсы и лендинги;
- `prototype/build_subpages.py` — доп. страницы и общая сборка;
- `prototype/tilda_*.html` — исходники блоков;
- `prototype/tilda_blocks_min/` — только результат минификации, руками не править.

Идея простая: один вход, один набор исходников, один артефакт на каждый блок.
