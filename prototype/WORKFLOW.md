# Рабочий контур

Для повседневной работы хватает трёх команд:

```bash
make build
make status
make all
```

Что они делают:

- `make build` — пересобирает минифицированные копии Tilda-блоков в `prototype/tilda_blocks_min/`;
- `make status` — показывает отсутствующие или устаревшие min-файлы;
- `make all` — делает build и затем проверку статуса.

Источники правим только в `prototype/*.html`.
Результат минификации хранится отдельно в `prototype/tilda_blocks_min/` и не редактируется вручную.
