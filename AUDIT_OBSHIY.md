## 1) Тех. SEO

### Текущее состояние

- `https://dymova-english.ru/robots.txt` доступен; закрыты служебные URL (`/lmsfoxinburg`, `/thanks`, `/policy`, `/oshibka`, Tilda-служебные пути), sitemap указан явно.
- `https://dymova-english.ru/sitemap.xml` содержит 14 боевых URL: `/`, `/doshkolniki`, `/mladshie-shkolniki`, `/podrostki`, `/reading`, `/grammar`, `/preparation`, `/online-zanyatiya`, `/podderzhivayushchie-online`, `/standartnye-offline`, `/letnyaya-akademiya`, `/kontakty`, `/novosti`, `/vakansii`.
- По живым страницам:
  - `lang="ru"` присутствует;
  - canonical есть на всех проверенных страницах;
  - `hreflang` не используется;
  - title и meta description уникальны на всех 14 страницах.
- Schema.org уже есть:
  - sitewide: `EducationalOrganization` / `LocalBusiness` + `WebSite` в `org_localbusiness.html`;
  - homepage: `FAQPage`;
  - `Course`: `/reading`, `/grammar`, `/preparation`, `/letnyaya-akademiya`;
  - `BreadcrumbList`: внутренние страницы из `prototype/seo_schema/breadcrumb_*.html`.

### Проблемы

- Нет `hreflang` — это не критично для одного языка, но оставляет без явной языковой декларации для будущих версий.
- Не хватает расширенной разметки:
  - `Event` для летних смен/интенсивов;
  - `AggregateRating` / `Review` для доверия;
  - `Person`/`Course`-усиления для преподавателей и продуктовых посадочных;
  - `FAQPage` на остальных коммерческих страницах, если FAQ реально есть в контенте.
- `YML`-фиды «Услуги и предложения в поиске» (сессия 28):
  - **Образование** (`feed_education.xml`): было заблокировано 2 из 4 предложений — в `param name="Продолжительность"` стояла недопустимая единица `занятие` (Яндекс принимает только час/день/месяц). Исправлено (чтение → 24 часа, грамматика → 30 часов), выложено на gh-pages, кэш CDN сброшен; Яндекс перепроверяет автоматически.
  - **Вакансии** (`feed_vacancies.xml`): 3 из 3 предложений заблокированы. Кроме техошибок (нет `price`/`currencyId`/`param name="Конверсия"`) есть стоп-факторы: формат требует минимум 25 наборов-сетов (у школы 3 вакансии), зарплату в каждой вакансии (у преподавателей нет числа) и страницы без попапов. Решение владельца: **оставить как есть** (будет висеть в ошибках).
  - Оговорка: формат «Образование» рассчитан на онлайн-курсы; часть курсов школы офлайн, поэтому ручная модерация может ограничить показ даже после автопроверки.

### Рекомендации

**P1**
- Проконтролировать, что education-фид после автопроверки снял блокировку 2 предложений. По вакансиям решение — оставить как есть (формат нереалистичен для локальной школы с 3 вакансиями).
- Поддерживать canonical только на канонических URL и не плодить дубли.

**P2**
- Добавить `Event` для смен Летней Академии.
- Добавить `AggregateRating` и `Review` только после накопления достаточного числа реальных отзывов.
- Приготовить шаблон FAQPage для ключевых лендингов.

**P3**
- Рассмотреть `hreflang` только если появятся языковые версии сайта.

#### План по /novosti (обсуждался в сессии 28, решение владельца отложено)

Сейчас `/novosti` — витрина из 2–3 карточек, уводящая в группу VK, из-за чего трафик и SEO-вес утекают со своего домена. Предложение:

1. **Развернуть логику:** статья живёт на сайте по своему URL (`/novosti/<slug>`), а в VK/Telegram/MAX идёт анонс со ссылкой (а не «полная лента в VK»).
2. **Два потока контента (микс 1:1):** события/трасты (смены Академии, дни открытых дверей, кейсы учеников) + полезное для родителей (SEO под informational-запросы, линкуется на коммерческие страницы и питает будущие лендинги).
3. **Календарь 2–4 поста/мес.** Заголовки-примеры: «Со скольки лет учить английский», «Как подготовиться к ОГЭ/ЕГЭ по английскому», «Репетитор или языковая школа», «Английский для взрослых с нуля», «Немецкий/китайский для школьников в Долгопрудном», кейс «как подтянул грамматику за курс».
4. **Технически:** отдельные URL постов, разметка `BlogPosting`/`Article` + `BreadcrumbList`, единый шаблон (обложка, 600–900 слов, FAQ, CTA «Бесплатная диагностика» + внутренние ссылки), лента с пагинацией.
5. **Дистрибуция каждого поста:** VK + Telegram + рассылка через MAX-бот + подпись в письмах.
